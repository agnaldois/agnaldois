# Agnaldo Souza

### **💻 Software Engineer**  
**Software Engineer with 4+ years of experience** working in large-scale, mission-critical systems, especially in the financial domain.

I specialize in backend development with Java and Spring Boot, building and evolving microservices that require reliability, traceability, and auditability. I have hands-on experience improving legacy systems and modernizing processes without disrupting business operations.

In the financial sector, I designed and implemented an AI-powered agent (Claude + Python, human-in-the-loop) to automatically document over 100 legacy Java EJB/WAR services. 106 services were mapped in just 3 weeks, reducing onboarding time from 2 weeks to 5 days and improving cross-team autonomy.

I also worked on fraud prevention systems, implementing backend services that processed approximately 5,000 phone-based interactions and contributed to an 80% reduction in fraud attempts, according to metrics assessed by the fraud prevention team. These solutions were built using Java and PostgreSQL, in close collaboration with business and compliance stakeholders.

Beyond coding, I value system design, observability, documentation, and pragmatic solutions that balance technical excellence with business impact.
